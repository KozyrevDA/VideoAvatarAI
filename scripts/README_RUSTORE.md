# Публикация в RuStore

## Быстрый запуск

```bash
cd scripts
pip install requests cryptography pyjwt Pillow
python3 publish_rustore.py
```

Скрипт создаст карточку приложения в RuStore **без** отправки на модерацию.

## Что делает скрипт

1. Генерирует JWT токен через RSA-256
2. Создаёт/обновляет карточку `org.nla.videoavataraii`
3. Добавляет описание на русском (полное + краткое + "что нового")
4. Загружает иконку 512×512
5. Загружает 4 скриншота 1080×1920
6. Загружает feature graphic 1024×500
7. **НЕ** отправляет на модерацию

## После скрипта нужно вручную в консоли

1. Открыть https://console.rustore.ru
2. Найти `org.nla.videoavataraii`
3. **Загрузить APK** (release build со своим keystore)
4. Указать цены на подписки и токены
5. Нажать «Отправить на модерацию»

## Безопасность ключа

**Обязательно** после первого использования:
1. Зайди в https://console.rustore.ru → Настройки → API ключи
2. Отзови старый ключ
3. Создай новый
4. Замени `PRIVATE_KEY_B64` в `publish_rustore.py`
5. Или передай через переменную окружения:

```bash
export RUSTORE_PRIVATE_KEY="новый_ключ_base64"
python3 publish_rustore.py
```
